# OpenThread CLI - TCAT Example

## Command List

- help [#help]
- start [#start]
- stop [#stop]

### help

print help

```bash
tcat help
help
start
stop
Done
```

### start

Start tcat server and ble advertisement.

```bash
tcat start
Done
```

### stop

Stop tcat server and ble advertisement.

```bash
tcat stop
Done
```
